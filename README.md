# BixAI - AI-Powered Desktop Assistant

A cross-platform desktop application that brings AI assistance to your fingertips through customizable keyboard shortcuts. BixAI integrates Google Gemini AI with a modern Flutter UI and FastAPI backend.

- **AI Chat Interface** - Real-time conversations with Google Gemini AI
- **Custom Keyboard Shortcuts** - Create personalized shortcuts for AI commands
- **Screenshot Analysis** - Analyze screenshots with AI-powered insights
- **License Management** - Integrated Lemon Squeezy licensing system
- **Request Tracking** - Monitor AI usage and statistics
- **Dark/Light Theme** - Beautiful UI with theme support
- **Cross-Platform** - Windows, macOS, and Linux support
